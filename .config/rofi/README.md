# Rofi with Dracula Theme
## Install Rofi
```bash
sudo pacman -S rofi
```
## Clone Dracula Theme
[Dracula Rofi Repo](https://github.com/dracula/rofi)<br/>
Select `config2.rasi`, rename it as `dracula.rasi
```bash
sudo mv dracula.rasi /usr/share/rofi/themes/
```
## Icons
The icons theme is `Papirus`
